# aikit

**Share your AI rules. Sync with your team.**  
**共享规则，协作开发。**

Unified CLI for sharing and syncing AI dev assets (Skills, Rules, MCP, Commands). Project rules in `.aikit.yaml`—version them, share with the team, one `aikit sync` and everyone is aligned.

- **Design (Chinese):** [docs/design-zh.md](docs/design-zh.md)
- **Code & i18n:** All source code, comments, and user-facing messages are in English for internationalization.

## Quick start

```bash
make build
./bin/aikit version
```

The binary is written to `bin/aikit`. Or: `go build -o bin/aikit .`
